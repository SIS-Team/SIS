Installation
------------

1. vom public_html einen Symlink zum Entwicklungsbereich.
2. in config.php RELATIVE_ROOT umstellen 
3. Startseite im Browser öffnen.

   - Cookies akzeptieren
   - Umleitung auf https : für tests direkt über ssl zugreifen.

4. icon-Pfad falsch.

4. Datenbank definieren.
